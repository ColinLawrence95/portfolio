# Portfolio Website

A modern, animated portfolio website built with React, TypeScript, and Framer Motion. Features smooth animations, responsive design, and showcases projects and technical skills.

## 🚀 Live Demo

[View Portfolio](https://colinlawrence.github.io/portfolio)

## ✨ Features

- **Animated UI Components** - Smooth animations powered by Framer Motion
- **Responsive Design** - Optimized for all screen sizes
- **Project Showcase** - Highlights key projects with live links
- **Skills Display** - Visual representation of technical skills
- **Modern Stack** - Built with React 19, TypeScript, and Vite

## 🛠️ Tech Stack

- **Framework:** React 19
- **Language:** TypeScript
- **Build Tool:** Vite
- **Animations:** Framer Motion
- **Icons:** React Icons
- **Styling:** CSS
- **Deployment:** GitHub Pages

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/ColinLawrence95/portfolio.git

# Navigate to project directory
cd portfolio

# Install dependencies
npm install
```

## 🏃 Development

```bash
# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Deploy to GitHub Pages
npm run deploy
```
